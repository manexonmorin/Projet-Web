Game Jam

Un game jam est un hackathon avec pour thème principal les jeux vidéo, mais pouvant aussi porter sur les jeux au sens large. Votre objectif dans ce concours est de créer un jeu vidéo avec un thème précis dans un temps limité. Les participants, groupés en équipes, doivent créer un jeu généralement le temps d'un week-end, sur 24 à 72 heures.
